## Overview

Building a home Security Operations Center is one of the best ways to develop
practical blue team skills outside of a corporate environment. This project
documents the architecture, tools, and lessons learned from building a
functional SOC on consumer hardware.

## Goals

- Centralized log collection and analysis
- Threat detection and alerting
- Threat intelligence integration
- Incident response practice
