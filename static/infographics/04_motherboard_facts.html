WHAT IT DOES
  • Connects CPU, RAM, GPU, PSU, storage, and peripherals
  • Transfers data via etched copper traces (the bus)
  • Houses BIOS/UEFI firmware chip for system init
  • Distributes regulated power from PSU to components
  • Provides expansion slots for add-in cards
  • Runs POST on boot to verify all hardware
THE SYSTEM BUS
  • Data bus — transfers actual data between components
  • Address bus — carries RAM memory locations
  • Control bus — sync signals and clock timing
  • Power rails — distributes 3.3V / 5V / 12V
  • Bus = Latin "omnibus" (for all)
  • Bus speed locked to system clock (MHz)
