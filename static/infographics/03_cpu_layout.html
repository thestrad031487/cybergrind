// SIMPLIFIED CPU ARCHITECTURE DIAGRAM
CORE (ALU + FPU)
Each core independently executes the fetch-decode-execute cycle. ALU handles arithmetic/logic; FPU handles floating-point math.
CACHE HIERARCHY
L1 (fastest, per-core) → L2 (larger, per-core) → L3 (shared across cores). Reduces latency vs accessing main RAM.
CONTROL UNIT (CU)
Decodes instructions from IR. Sends control signals to ALU, FPU, memory, and I/O. Drives the fetch-execute cycle.
MEMORY CONTROLLER
Integrated Memory Controller (IMC). Manages read/write to RAM. Modern CPUs integrate this on-die for lower latency.
BRANCH PREDICTOR
Guesses the next instruction path before a branch resolves. Critical for pipeline efficiency in modern out-of-order CPUs.
PCIe / I/O CONTROLLER
Manages high-speed data lanes to GPU, NVMe storage, and other PCIe devices. Integrated on modern CPUs.
CPU PINS / LGA PADS
Physical connection to motherboard socket. LGA pins are on the board; AM5/LGA1700 have hundreds of contacts for power and data.
