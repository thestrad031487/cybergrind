// ATX MOTHERBOARD — COMPONENT MAP
CPU SOCKET
LGA/AM5 socket with hundreds of contact pads. Holds CPU and connects via VRM-regulated power.
RAM SLOTS (DIMM)
DDR5 slots for system memory. A1/B1 = Channel A/B for dual-channel bandwidth.
BIOS/UEFI CHIP
Firmware that runs POST at boot and initializes hardware before the OS loads.
POWER CONNECTORS
24-pin ATX (main board power) + 8-pin EPS (CPU power). Both required for operation.
PCIe x16 SLOTS
Primary slot for GPU at full x16 bandwidth. Secondary slot often wired at x4 via chipset.
M.2 SLOTS
High-speed NVMe SSD slots. PCIe 5.0 x4 = ~14 GB/s. CPU-direct for lowest latency.
CHIPSET (PCH)
Platform Controller Hub. Routes USB, SATA, extra PCIe, and secondary M.2 through a DMI link to CPU.
REAR I/O PANEL
USB-A/C, HDMI/DP (integrated GPU), Ethernet, WiFi antenna, audio jacks, BIOS flash button.
SATA PORTS
6 Gb/s ports for SSDs and HDDs. Managed by the PCH/chipset, not directly CPU-attached.
