Methodology

How data flows through CyberGrind

Update Cadence

Each source is collected on its own schedule — from hourly (URLhaus IOCs) to daily (CISA KEV, EPSS). Update frequencies are documented per-source above and reflected in the tool UI where relevant.

Caching Strategy

Cloudflare Workers cache upstream API responses at the edge to stay within free-tier rate limits and reduce latency. Cache TTLs range from 1 hour (OSINT feeds) to 6 hours (AbuseIPDB blacklist).

Known Limitations

All data is sourced from free tiers or public feeds. Coverage is not comprehensive. AbuseIPDB and VT quotas reset at midnight UTC. NVD enrichment can lag CVE publication by days to weeks.

AI-Generated Content

Daily CyberNews commentary is generated by a local Llama 3.2 (3B) model. It is editorial assistance only — always verify claims against the linked source headlines. The model operates offline; no data is sent to external AI services.

Data Storage

The CTI pipeline stores collected data in a local SQLite database on the Windows workstation hosting api.cybergrind.org. No personally identifiable information is stored. Data is for display and research purposes only.

Infrastructure

The site is hosted on Cloudflare Pages. The API backend runs in Docker on a Windows workstation exposed via Cloudflare Tunnel. Collector workers are scheduled Python scripts running inside Docker. The tunnel requires --protocol http2 due to VPN constraints.

Licensing & Attribution

Third-party API & Data Attribution

CyberGrind uses a mix of U.S. government public domain data (CISA KEV, NVD), open-source community feeds (abuse.ch, IPsum, Emerging Threats), and commercial free-tier APIs (AbuseIPDB, VirusTotal, NewsAPI). Each source is used in accordance with its respective terms of service.

abuse.ch (URLhaus, MalwareBazaar, Feodo Tracker) is used under their non-commercial free use policy. Attribution is provided in the tool UI. See abuse.ch for licensing details.

MITRE ATT&CK® is used under the ATT&CK Terms of Use. ATT&CK® is a registered trademark of The MITRE Corporation.

AbuseIPDB data is community-contributed and used under their Terms of Service. Attribution is provided in the Threat Map UI.

VirusTotal is accessed via their public API under the VirusTotal Terms. CyberGrind does not store, redistribute, or commercialize VirusTotal data.

CyberGrind is a non-commercial personal portfolio and educational resource. No data collected through these sources is sold, redistributed, or used for purposes outside of site functionality and security education.
